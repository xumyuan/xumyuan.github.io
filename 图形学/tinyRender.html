直线光栅化 ​

给定两点v0(x0, y0)和v1(x1, y1)，我们可以通过Bresenham算法来计算出这两点之间的所有像素点。Bresenham算法是一种直线光栅化算法，它的基本思想是利用递增的误差来决定下一个像素点的位置。这个算法的优点是只需要整数运算，而且不需要使用浮点数运算，因此效率很高。 具体的做法是：

  1. 先确定x,y轴变化大的那个轴，假设是x轴（若不是则交换）。
  2. 计算斜率k,k=ΔyΔx
  3. 遍历x轴上的像素点，对于每一个x，x增加1，y增加k，然后判断y是否需要增加1，累积y的变化率error，当error大于0.5时，y增加1，error减1。
cpp
void Bresenham(int x0, int y0, int x1, int y1) {
    int dx = x1 - x0;
    int dy = y1 - y0;
    int error = 0;
    int y = y0;
    int k = dy / dx;
    for (int x = x0; x <= x1; x++) {
        drawPixel(x, y);
        error += k;
        if (error > 0.5) {
            y++;
            error -= 1;
        }
    }
}

但是这样，还用到了浮点数运算，可以通过乘以dx来避免浮点数运算。

cpp
void Bresenham(int x0, int y0, int x1, int y1) {
    int dx = x1 - x0;
    int dy = y1 - y0;
    int error = 0;
    int y = y0;
    int k = dy;
    for (int x = x0; x <= x1; x++) {
        drawPixel(x, y);
        error += k;
        if (error > dx) {
            y++;
            error -= 2*dx;
        }
    }
}
